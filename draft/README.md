# IETF Internet-Draft

The ApertoMemory format is specified in an IETF Internet-Draft.

**Current version: draft-ferro-apertomemory-01**
https://datatracker.ietf.org/doc/draft-ferro-apertomemory/

Changes in -01: expanded Security Considerations (nonce management and
AEAD limits, key rotation, device and key revocation) and an
Implementation Status section documenting two interoperable
implementations. No changes to the wire format, the CDDL schema, or
the test vectors: implementations of -00 remain conforming.

Sources in this folder: `draft-ferro-apertomemory-01.xml` (canonical,
RFC 7991 v3) and the compiled `.txt`.
