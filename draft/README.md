# Internet-Draft

Working copy of the ApertoMemory Internet-Draft
(`draft-ferro-apertomemory-00`): the normative specification of the
format — data model (CDDL), cryptographic envelope, key hierarchy,
export container, and security considerations.

**Status: not yet submitted to the IETF Datatracker.** This section
will link to the Datatracker entry once the draft is submitted.

Built and validated with the official `xml2rfc` toolchain:

```bash
pip install xml2rfc
xml2rfc draft-ferro-apertomemory-00.xml --text --html
```
